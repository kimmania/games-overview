Simple Puzzle Games

Mobile-first browser puzzles and card games built with TypeScript. Each app installs as a PWA and saves progress locally.

Logic Puzzles

Cage Logic

New

KenKen-style · cage arithmetic · multiple sizes

Play Source

Concept

A KenKen-style logic puzzle: fill the grid so every row and column contains each digit once, and every cage computes to its target using the given arithmetic operation.

Cryptogram

Substitution cipher · famous quotes · three lives

Play Source

Concept

A classic cryptogram: each puzzle is a famous quote where every letter A–Z has been replaced by a different letter. Punctuation and spaces stay as they are. Decode the substitution to reveal the original quote.

Hitori

Shade duplicates · connectivity · grid size sets difficulty

Play Source

Concept

A Nikoli-style elimination puzzle: the grid already shows numbers. You shade some cells black so that shaded cells are “removed” from their row and column, while obeying three global rules at once.

Kakuro

Crossword-style runs · combination clues · three difficulties

Play Source

Concept

Often described as a crossword with numbers: fill white cells with digits 1–9 so each horizontal or vertical run adds up to its clue. No digit repeats within a single run.

Number Sums

Rectangular grids · colored regions · four difficulties

Play Source

Concept

A logic puzzle where numbers sit in a grid divided into colored regions. You decide which numbers count toward the puzzle by circling them (or crossing out numbers you exclude). Every row, column, and region has a target sum shown on the edge of the grid.

Sightline

New

Kurodoko · visibility counts · block shading

Play Source

Concept

A Kurodoko shading puzzle: shade cells so every number “sees” exactly as many cells as its count in all four directions, while shaded cells block the line of sight and cannot touch orthogonally.

Star Map

New

Star Battle · region partitioning · three difficulties

Play Source

Concept

A spatial logic puzzle: place exactly two stars in every row, column, and region without them touching. Auto-Assist tracks deductions and flags impossible placements as you play.

Sudoku

9×9 grid · pencil notes · 40,000 puzzles

Play Source

Concept

The classic number-placement puzzle: fill a 9×9 grid so every row, column, and 3×3 box contains the digits 1–9 exactly once. Some cells are given as clues; you deduce the rest.

Trailhead

New

Number trail · Hamiltonian path · draw-to-solve

Play Source

Concept

A number-sequence trail puzzle: place numbers 1–N so every consecutive pair forms a valid step through the grid (4-way or 8-way adjacency). Givens are fixed clues and cannot be overwritten. A unique Drag mode lets you draw the trail directly — the game auto-fills the sequence as you trace the path.

Card Games

Solitaire

One PWA · 17 rule sets · switch games from the header menu

Play Source

Concept

A single installable card-game app with many classic solitaire variants. Tap Games in the header to switch rule sets; each game keeps its own saved layout in local storage. Touch-friendly drag-and-drop, undo, and automatic win detection are shared across variants.

Games included

  • Baker's Dozen — Thirteen columns; kings start at the bottom; build down in suit one card at a time.
  • Beleaguered Castle — Aces start on foundations, kings in reserve; build tableau down in suit into eight columns.
  • Canfield — Reserve pile, cyclic foundations from a starter rank; tableau builds down alternating color.
  • Clock — Turn cards onto their hour pile; win before the fourth king reaches the center.
  • Easthaven — Seven columns, build down in suit; one card at a time from the stock.
  • Eight Off — Eight cells and columns; build tableau down in suit, foundations up.
  • Forty Thieves — Two decks; build tableau down in suit; fill eight foundations Ace to King.
  • FreeCell — Use four free cells to build foundations Ace to King.
  • Klondike — Classic solitaire; build foundations Ace to King (draw-1, unlimited stock recycle).
  • Monte Carlo — Remove pairs of matching rank that touch; redeal when stuck.
  • Pyramid — Pair exposed cards that sum to 13; clear the pyramid and stock.
  • Scorpion — Build down in suit on seven columns; clear four King-to-Ace runs.
  • Spider (1 suit) — Easiest; build descending stacks; suit does not matter.
  • Spider (2 suits) — Medium; build down in one suit using spades and hearts only.
  • Spider (4 suits) — Hardest; build down in matching suit with all four suits.
  • TriPeaks — Clear three peaks; play exposed cards one rank above or below the waste.
  • Yukon — All cards face up; move any face-up stack; build alternating colors.
