Simple Puzzle Games

Mobile-first browser puzzles built with TypeScript. Each game installs as a PWA and saves progress locally.

Number Sums

Play Source

Rectangular grids · colored regions · four difficulties

Concept

A logic puzzle where numbers sit in a grid divided into colored regions. You decide which numbers count toward the puzzle by circling them (or crossing out numbers you exclude). Every row, column, and region has a target sum shown on the edge of the grid.

How to play

  1. Read the target totals for each row (left), column (top), and region (badge).
  2. Tap cells in Circle mode to include a number, or Erase mode to exclude it.
  3. A target turns green when your circled numbers in that line add up correctly — check every line before you finish.
  4. Win when every cell is marked and all row, column, and region sums match their targets.

Sudoku

Play Source

9×9 grid · pencil notes · 40,000 puzzles

Concept

The classic number-placement puzzle: fill a 9×9 grid so every row, column, and 3×3 box contains the digits 1–9 exactly once. Some cells are given as clues; you deduce the rest.

How to play

  1. Choose a difficulty (Easy through Expert) and start a new puzzle.
  2. Select an empty cell, then tap a digit on the number pad to enter it.
  3. Use Notes mode for pencil marks when multiple candidates are possible; Fill Notes can auto-fill candidates on empty cells.
  4. Avoid contradictions — the app tracks mistakes against the hidden solution. Undo, reset, and resume are available anytime.

Kakuro

Play Source

Crossword-style runs · combination clues · three difficulties

Concept

Often described as a crossword with numbers: fill white cells with digits 1–9 so each horizontal or vertical run adds up to its clue. No digit repeats within a single run. Black clue cells show the target sum for the run below or to the right.

How to play

  1. Tap a light (fillable) cell, then choose a digit. The top-left number on a clue cell is the down sum; the bottom-right is the right sum.
  2. Use pencil Notes for candidates; impossible notes are pruned as you place digits.
  3. Open the Cheatsheet to see which digit combinations are valid for a given run length and sum.
  4. Complete every run without repeats to solve the puzzle.

Hitori

Play Source

Shade duplicates · connectivity · grid size sets difficulty

Concept

A Nikoli-style elimination puzzle: the grid already shows numbers. You shade some cells black so that shaded cells are “removed” from their row and column, while obeying three global rules at once.

How to play

  1. No duplicates — in each row and column, every number appears at most once among unshaded cells.
  2. No touching shades — shaded cells cannot share an edge (diagonal is fine).
  3. Stay connected — all unshaded cells must form one group via orthogonal moves.
  4. Tap a cell to cycle: undecided → white ring (keep) → black (shade) → undecided. Grid size sets difficulty (5×5 Easy up to 12×12 Expert).
